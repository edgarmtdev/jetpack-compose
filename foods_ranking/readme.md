# FoodRank

A modern Android app built with Jetpack Compose as part of my Android learning journey.

This project focuses on:
- MVVM architecture
- StateFlow & reactive UI
- Jetpack Compose
- Dynamic ranking systems
- Compose animations
- Modern Android development practices

---

## Features

- Like system
- Real-time food ranking
- Animated list reordering
- Async image loading
- Reactive state management

---

## Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- MVVM
- StateFlow
- Coroutines
- Coil 3
